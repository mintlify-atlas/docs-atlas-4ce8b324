Compose Performance Skills is a community library of Agent Skills built for Jetpack Compose performance work. Each skill is a focused `SKILL.md` file that an AI agent loads on demand to diagnose and fix a specific performance problem — from scroll jank and stability regressions to hot reload setup and CI enforcement. The library covers the full performance loop: Measure, Diagnose, Fix, and Verify.

## Get started

<Steps>
  <Step title="Clone the repository">
    Clone compose-performance-skills to a stable location on your machine.

    ```bash
    git clone https://github.com/skydoves/compose-performance-skills.git \
      ~/.claude/skills-sources/compose-performance-skills
    ```
  </Step>
  <Step title="Run the install script">
    The install script symlinks each skill into the flat layout your agent runtime expects.

    ```bash
    ~/.claude/skills-sources/compose-performance-skills/scripts/install-skills.sh
    ```
  </Step>
  <Step title="Restart your agent">
    Restart Claude Code (or your agent runtime). The skills are now active and will load automatically when you describe a Compose performance symptom.
  </Step>
  <Step title="Describe a symptom">
    Mention a Compose performance problem in your prompt. The agent matches trigger vocabulary from skill frontmatter and loads the relevant `SKILL.md` automatically.

    > "My LazyColumn drops frames during fast scroll. Recomposition counts spike on every scroll tick."
  </Step>
</Steps>

<Note>
  The install script is idempotent. Run it again after a `git pull` to pick up new skills without creating duplicate symlinks.
</Note>
